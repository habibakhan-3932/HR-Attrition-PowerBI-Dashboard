# HR-Attrition-PowerBI-Dashboard

Project Overview

This project is an HR Analytics Dashboard built using Microsoft Power BI to analyze employee attrition patterns and workforce insights.
The goal of this dashboard is to help organizations understand why employees leave, identify key trends, and support better HR decision-making.

Objectives

Analyze employee attrition rate

Identify departments with highest attrition

Understand gender and age-group trends

Evaluate job roles with higher turnover

Provide quick KPIs for HR managers

Tools & Technologies

Microsoft Power BI

Power Query – Data Cleaning & Transformation

DAX (Data Analysis Expressions) – Measures & Calculations

CSV Dataset – HR Employee Attrition Data

Dataset Information

The dataset contains employee-level information such as:

Age

Gender

Department

Job Role

Monthly Income

Years at Company

Overtime

Job Satisfaction

Attrition (Yes/No)

Key KPIs

Total Employees

Attrition Rate (%)

Average Salary

Average Age

Dashboard Features

Attrition by Department

Attrition by Gender

Attrition by Age Group

Attrition by Job Role

Overtime vs Attrition

Interactive visuals with filters

Data Cleaning Steps

Removed unnecessary columns (EmployeeCount, StandardHours, Over18)

Checked data types

Removed duplicate employee records

Renamed table for clarity

Insights Gained

Departments with higher turnover can be identified quickly.

Certain job roles show higher attrition trends.

Overtime has a visible impact on employee retention.

Age and gender distributions help in workforce planning.

How to Use

Download the .pbix file.

Open with Power BI Desktop.

Interact with filters and visuals to explore insights.
